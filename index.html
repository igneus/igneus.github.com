My work online

  • Software projects
    • Church Calendar API and calendarium-romanum

      Ruby gem computing Roman Catholic liturgical calendar and an API offering it's fruits on-line.

    • E-Antiphonal

      Web application proxying another web (Czech Liturgy of the Hours on-line) and extending it's content with my musical settings of the rendered texts - thus turning a "breviary" in an "antiphonal".

    • gly

      A simple language encoding Gregorian chant and a tool translating it to another similar language.

    • grely

      Convertor of two text-based musical formats (Gregorio GABC to LilyPond). Leverages a PEG-parser of the GABC language.

    • pslm

      Automates pointing of psalm texts for various chant tunes. Supports several output formats.
